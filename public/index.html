how-to

clicking on the card flips from front to back

incorrect will move the card to the back of the current stack

correct will remove the card from the current stack

what correct and incorrect mean is up to you

when the current stack is emtpy, you will be alerted and it will reset

switch between stacks using the select chapter radio buttons

the Reverse Mode checkbox toggles Reverse Mode, displaying the backs of cards first

about

sam raker made this website to help him learn বাংলা (bangla/bengali.)

bangla vocabulary and english translation in the radice stacks are taken from Complete Bengali by William Radice. no copyright is claimed.

vocab in the mango chapters is adapted from the mango languages bengali course. no copyright is claimed

translation of terms in the misc chapter was done using google translate and/or wiktionary.

any mistakes are the fault of sam raker. please let him know if you see any. also, feel free to ask if you're confused by the nomenclature used in the pronouns chapter.

fork the repo and replace the front and back of the objects in cards.mjs (you'll probably want to change the title too)
